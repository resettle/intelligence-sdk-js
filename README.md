# Resettle Intelligence SDK for JavaScript

## Installation

The packages can be installed separately with

```bash
npm install @resettle/place-api-sdk
```

or

```bash
pnpm add @resettle/place-api-sdk
```

or

```bash
yarn add @resettle/place-api-sdk
```

based on your need.

## OpenAPI Specs

You can view all the specifications at `openapi/**.json`.
