# @resettle/occupation-api-sdk

A TypeScript SDK client for the Resettle Intelligence occupation API.

## Usage

You can install the SDK package from npm

```bash
npm install @resettle/occupation-api-sdk
```

If you are using yarn as the package manager, then

```bash
yarn add @resettle/occupation-api-sdk
```

If you are using pnpm as the package manager, then

```bash
pnpm add @resettle/occupation-api-sdk
```

Next, you can visit [Resettle Developer](https://resettle.ai/dev) to get a free API key to test it out. You can also checkout the [API documentation](https://resettle.github.io/doc#occupation)
